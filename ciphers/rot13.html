ROT13 Cipher

Simple letter substitution cipher that replaces each letter with the letter 13 positions after it

ROT13 (rotate by 13 places) is a special case of the Caesar cipher, where each letter is shifted 13 positions in the alphabet. Since the English alphabet has 26 letters, ROT13 is its own inverse - applying it twice returns the original text. This property makes it a popular method for hiding text (like spoilers or puzzle solutions) that readers can easily decode. ROT13 is not meant for security but for obscuring text from casual glances. It's widely used in online forums and has become a part of internet culture.

Security Warning: This cipher provides minimal security and is easily broken with modern techniques. It should only be used for educational purposes or when security is not a concern.
